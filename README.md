# EduSmart - Student Management System

A premium, modern, and responsive Student Management System built with Node.js, Express, and MySQL.

## Features
- ✨ **Premium UI**: Glassmorphism design with dark mode and smooth animations.
- 📊 **Dashboard Stats**: Real-time overview of total students, top graders, and recent enrollments.
- 📝 **Full CRUD**: Add, Edit, View, and Delete student records.
- 🔍 **Live Search**: Instant filtering by name or course.
- 📱 **Responsive**: Optimized for desktops, tablets, and mobile devices.

## Tech Stack
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Backend**: Node.js, Express.js
- **Database**: MySQL
- **Icons/Fonts**: Font Awesome, Google Fonts (Outfit)

## Quick Start
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run `schema.sql` in your MySQL server.
4. Update `.env` with your MySQL credentials.
5. Start the server: `npm start`
6. Open `http://localhost:5000`
